# AI-Driven Lunar Soil Composition Analysis Using Imagery and Large Language Models

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Python](https://img.shields.io/badge/python-3.8%2B-blue.svg)

AI-driven analysis of lunar soil composition using satellite imagery and Large Language Models (LLMs). Combines vision models and LLM reasoning to classify regolith, map minerals, and generate explainable reports for lunar exploration and resource planning.

This project leverages state-of-the-art computer vision models and Large Language Models to analyze lunar soil composition from satellite imagery. The system provides:

- **Automated Regolith Classification**: Deep learning models to classify different types of lunar soil
- **Mineral Mapping**: Identification and localization of key minerals in lunar samples
- **LLM-Powered Analysis**: Intelligent interpretation and explanation of composition data
- **Explainable Reports**: Human-readable reports generated by LLMs for scientific and mission planning purposes

## ğŸ› ï¸ Installation

### Prerequisites

- Python 3.8 or higher
- CUDA-capable GPU (recommended for training)
- API keys for LLM providers (OpenAI, Anthropic, etc.)

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Safwan2003/AI-Driven-Lunar-Soil-Composition-Analysis-Using-Imagery-and-Large-Language-Models.git
   cd AI-Driven-Lunar-Soil-Composition-Analysis-Using-Imagery-and-Large-Language-Models
   ```

2. **Create a virtual environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Install the package in editable mode**
   ```bash
   pip install -e .
   ```

5. **Configure environment variables**
   ```bash
   cp .env.example .env
   # Edit .env and add your API keys
   ```

6. **Update configuration**
   Edit `config.yaml` to customize settings for your use case.

## ğŸ¯ Quick Start

### 1. Data Preparation

```python
from data import DataLoader, Preprocessor

# Load and preprocess lunar imagery
loader = DataLoader(config_path="config.yaml")
data = loader.load_images("datasets/raw")

preprocessor = Preprocessor()
processed_data = preprocessor.process(data)
```

### 2. Model Training

```python
from models import LunarSoilClassifier

# Initialize and train the model
model = LunarSoilClassifier(config_path="config.yaml")
model.train(processed_data)
model.save("models/saved_models/classifier.pth")
```

### 3. LLM Integration

```python
from llm_integration import LLMAnalyzer

# Analyze results with LLM
analyzer = LLMAnalyzer(provider="openai", model="gpt-4")
interpretation = analyzer.analyze(predictions, context="lunar_mission_planning")
```

### 4. Generate Reports

```python
from visualization import ReportGenerator

# Create comprehensive reports
generator = ReportGenerator()
generator.create_report(
    predictions=predictions,
    interpretations=interpretation,
    output_path="reports/analysis_report.pdf"
)
```
